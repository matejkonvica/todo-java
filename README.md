Naklonujte repozitář nebo stáhněte zdrojové kódy.

Spusťte aplikaci pomocí Maven: mvn spring-boot:run

Alternativně lze aplikaci spustit přímo z IDE spuštěním hlavní třídy aplikace, kterou můžete najít na src/main/java/cz/vse/todo/todo_java/TodoJavaApplication.java.

Po spuštění aplikace je server dostupný na adrese: http://localhost:8080

Heslo na přístup k aplikace je Password. Hodnotu lze změnit v konfiguračním souboru application.yml nalezený na src/main/resources/application.yml
